# Smart Travel Planner (Express + EJS)

This project was converted from a static HTML/CSS/JS site into a server-rendered Node.js app using Express + EJS.

## Run locally

```powershell
npm install
npm start
```

Then open `http://localhost:3000`.

## Dev mode (auto-reload)

```powershell
npm install
npm run dev
```

## Project structure

- `server.js`: Express server
- `views/index.ejs`: Server-rendered page
- `public/`: Static assets (CSS/JS)
